## 2. Resolved High-Impact Issues

### 🔴 The "Unauthorized" (401) Error
*   **Issue**: Backend was rejecting tokens with "invalid algorithm".
*   **Cause**: We were using `HS256` (symmetric) with a secret, but the tokens were `ES256` (asymmetric).
*   **Fix**: Installed `jwks-rsa` and configured `JwtStrategy` to use `secretOrKeyProvider` for dynamic public key fetching.

### 🔴 Admin Login Loop
*   **Issue**: Admin couldn't log in (loading loop).
*   **Cause**: Cookie name mismatch (`sb-auth-token` vs `sb-admin-auth-token`) between middleware and server client.
*   **Fix**: Standardized `sb-admin-auth-token` across all admin-panel auth logic.
